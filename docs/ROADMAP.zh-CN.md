# AllClaws 研究路线图

> 2026 年下半年战略优先级和研究重点领域。

## 2026 年下半年优先事项

### 主题：个人与企业分叉

2026 年下半年的主要重点是记录和分析**个人力量倍增器**与**企业自动化**AI 代理范式之间正在出现的分叉。

**关键问题：**
1. 这些范式在架构上如何不同？
2. 开发者在选择其中一种时做出什么权衡？
3. 生态系统会收敛还是进一步分歧？
4. 什么模式可以在范式之间转移？

## 2026 年第三季度（7-9 月）

### 1. 完成统一平台比较

**状态：** 进行中
**目标：** 完成所有 20 个平台的架构文档

**交付成果：**
- [ ] `architecture/platform_comparison.md` — 所有 20 个平台的统一比较
- [ ] 每个平台的标准格式（分类、架构、图表）
- [ ] 跨平台比较矩阵
- [ ] 中文翻译（`platform_comparison.zh-CN.md`）

**范围：**
- **Claw 生态（13 个）：** OpenClaw、ClawTeam、GoClaw、IronClaw、Maxclaw、NanoClaw、Nanobot、ZeroClaw、HiClaw、QuantumClaw、Hermes-Agent、RTL-CLAW、Claw-AI-Lab
- **外部框架（7 个）：** SmolAgents、LangGraph、mcp-agent、CrewAI、AutoGen、Swarms、OpenAgents

### 2. MCP 生态系统深度分析

**状态：** 计划中
**目标：** MCP 协议采用和抵制的综合分析

**研究问题：**
- MCP 在生产部署中的实际 token 开销是多少？
- 哪些 MCP 服务器被广泛采用？
- MCP 原生框架（mcp-agent）与 MCP 适配器方法有何不同？
- 20 个跟踪平台的 MCP 采用率是多少？

**交付成果：**
- MCP 采用报告（原生 vs 适配器 vs 抵制）
- Token 成本分析
- MCP 服务器生态系统目录
- MCP 评估建议

### 3. 企业治理框架分析

**状态：** 计划中
**目标：** 记录新兴的企业 AI 代理治理方法

**研究范围：**
- Okta 的 AI 代理身份框架
- 企业部署模式（HiClaw、GoClaw、LangGraph、Swarms）
- 人在环路工作流
- 凭证隔离模式

**交付成果：**
- 企业治理比较矩阵
- 最佳实践文档
- 风险分析框架

## 2026 年第四季度（10-12 月）

### 1. 个人力量倍增器案例研究

**状态：** 计划中
**目标：** 记录 1PC（一人公司）成功故事

**研究范围：**
- 采访使用 AI 代理的独立创始人
- 记录工作流程和工具链
- 分析成本/收益与传统雇佣的对比
- 识别 1PC 环境中的平台偏好

**交付成果：**
- 5-10 个案例研究
- 共同模式分析
- 1PC 平台建议
- ROI 分析

### 2. 运行时性能基准测试

**状态：** 计划中
**目标：** 扩展测试框架以进行运行时性能指标测量

**当前状态：** 测试框架涵盖静态分析（177 个测试，165 个通过）
**目标状态：** 添加运行时性能指标

**要跟踪的指标：**
- 冷启动时间
- 内存使用（空闲和活跃）
- 响应延迟（p50、p95、p99）
- Token 效率（输出/输入比率）

**交付成果：**
- 20 个平台的性能比较
- 基准测试方法文档
- 性能回归检测

### 3. "自我改进"声明验证系列

**状态：** 计划中
**目标：** 自主学习声明的系统性验证

**继 Hermes 分析之后（2026 年 5 月）：**

**要验证的平台：**
- 任何声称"自我改进"能力的平台
- 基于强化学习的训练系统
- 自适应技能系统
- 性能优化声明

**方法论：**
- 源代码分析
- 文档与实现比较
- 随时间推移的实际性能改进证据

## 未来方向（2027 年及以后）

### 潜在重点领域

1. **移动优先代理** — 针对移动部署优化的 AI 代理
2. **边缘计算代理** — 资源受限的代理部署
3. **区块链集成代理** — 具有区块链身份/协调的代理
4. **垂直特定分析** — 医疗保健、金融、法律 AI 代理
5. **代理互操作性** — 跨平台代理通信协议

### 扩展标准

新平台可能会根据以下条件添加：
- 活跃开发（当年有提交）
- 战略意义（新颖模式、参考实现）
- 社区吸引力（stars、forks、讨论）
- 架构独特性（代表不同方法）

---

## 非目标

为了保持专注，AllClaws 明确**不**计划：

1. **对平台进行评级** — 没有"最佳平台"排名
2. **提供教程** — 其他来源涵盖操作指南
3. **提供咨询** — 我们研究，不提供建议部署
4. **预测市场** — 我们分析技术，而不是投资潜力
