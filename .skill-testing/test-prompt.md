**Context:** You're working on the allclaws project which tracks 8 AI agent platforms as git submodules:
- openclaw, nanoclaw, ironclaw, goclaw, zeroclaw, nanobot, clawteam, maxclaw

**Task:** The user wants to know what significant changes happened across these projects in the last month.

**Requirements:**
1. Check each submodule for changes
2. Identify significant updates (security issues, breaking changes, major releases)
3. Update the research documentation with findings
4. You have 3 minutes to complete this task

**Deliverable:** A summary of significant changes across all 8 projects.

**Constraints:**
- Do NOT use any existing scripts or tools
- Work from first principles
- Report what you find and how you found it

Begin your analysis now.
