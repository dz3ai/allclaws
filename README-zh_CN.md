# AllClaws：个人 AI 代理生态系统分析与测试

**AllClaws** 现在是早期研究阶段，方案还在动态调整，成熟后会拆成多个专项开发。专注于分析、比较和测试个人 AI 代理平台。这个伞形项目将架构分析、性能基准测试和个人 AI 助手领域的思想领导力结合在一起。

## 🎯 使命

通过以下方式推进个人 AI 助手领域：
- **分析**现有平台的架构和设计决策
- **基准测试**不同实现的功能和性能
- **开发**用于客观比较的测试框架
- **分享**通过技术写作和文档获得的见解

### 1. 架构分析与比较
**状态：** ✅ 积极开发

### 2. 个人代理测试框架
**状态：** ✅ 核心实现完成

一个全面的测试框架，用于跨多个维度评估个人 AI 代理：

**框架特性：**

## 🏗️ 技术架构

### 测试框架设计原则
- **安全优先**：加密凭据、权限验证、审计日志
- **TDD 方法**：测试驱动开发，首先编写失败的测试
- **多平台**：不同代理运行时的统一接口
- **可扩展**：新测试类型和平台的插件架构

### 关键技术
- **Bash 脚本**：核心执行和验证逻辑
- **JSON 配置**：人类可读的代理定义
- **JQ 处理**：高级 JSON 操作和验证
- **基于 Git 的版本控制**：安全、可审计的开发工作流

## 🚀 快速开始

### 架构分析
```bash
# 阅读全面的平台比较
cat architecture_comparison.md

# 查看中文翻译
cat architecture_comparison.zh-CN.md
```

### 测试框架
```bash
cd test_framework

# 设置框架
./scripts/setup.sh

# 验证代理配置
./scripts/validate_agent.sh agents/example_agent.json

# 运行安全测试
bash tests/test_security_privileges.sh

# 运行代理验证测试
bash tests/test_agent_validation.sh
```
